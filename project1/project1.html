Project 1: Exchange Rate Predictor

Project Overview

This is an exchange rate predictor using machine learning with PyTorch. Its purpose is to predict exchange rates 3-5 days in advance.

Implementation Approach

I did this by creating a transformer and LSTM model to analyze historical forex data and make predictions.

Key Features

  • Predicts exchange rates 3-5 days in advance
  • Uses PyTorch deep learning framework
  • Implements Transformer architecture
  • Implements LSTM neural network

Model Architecture

Transformer Model
Uses attention mechanisms to capture long-term dependencies in forex data
LSTM Model
Leverages sequential memory to learn patterns in currency exchange rates
Training Data
Historical forex data spanning multiple currency pairs and time periods
