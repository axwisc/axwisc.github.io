Experience 1: InVisit

Company Overview

InVisit is a company that creates a visitor management cloud solution for enterprises. It allows for guests to check in and check out through the use of their software suite.

My Role

I worked on the kiosk device that allows users to self sign in on a tablet. This was an important part of the visitor management system.

Key Responsibilities

  1. Developing the kiosk interface
  2. Implementing self check-in functionality
  3. Testing on tablet devices
  4. Integrating with cloud backend

Technologies Used

Frontend
Built user-friendly kiosk interface for tablet devices
Cloud Integration
Connected kiosk to visitor management cloud solution
Hardware
Worked with tablet devices for self-service check-in
